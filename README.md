# 한글 타자연습

한컴타자처럼 자리연습과 낱말연습을 진도별로 진행할 수 있는 간단한 웹앱입니다.

## 기능

- 진도별 자리연습: `ㅁㄴㅇㄹㅓㅏㅣ;`부터 시작해 누적 자판을 연습합니다.
- 진도별 낱말연습: 현재 단계까지 배운 키로 입력 가능한 단어만 자동으로 골라냅니다.
- 짧은 글 연습: 현재 단계까지 배운 키로 입력 가능한 짧은 문장을 연습합니다.
- 키보드 배열 표시, 운지법 오버레이 표시를 각각 켜고 끌 수 있습니다.
- 버튼형 단계 선택, 제한 시간, 목표 횟수, 연습 범위를 조절할 수 있습니다.
- 단계마다 작은 키보드 지도를 보여 주어 어떤 자판 위치를 배우는지 바로 알 수 있습니다.
- 화면 아래 큰 키보드에서 이번 단계, 이미 배운 키, 나중 단계 키를 색으로 구분합니다.
- 물음표 아이콘 위에 마우스를 올리거나 키보드로 선택하면 각 기능 설명을 볼 수 있습니다.
- 정확도, 타수, 진행률, 남은 시간을 계속 확인할 수 있습니다.

## 실행

```bash
npm install
npm run dev
```

## 배포

저장소를 GitHub에 올린 뒤 Pages 설정에서 Source를 `GitHub Actions`로 지정하면, `main` 또는 `master` 브랜치에 push할 때 자동 배포됩니다.
